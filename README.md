# OptimusTech

![OptimusTech Logo](./imgs/logo.png)

## 🚀 Sobre o Projeto

O **OptimusTech** é uma landing page desenvolvida como parte do desafio **7DaysOfCode** da Alura. O objetivo é criar uma página inicial para uma empresa fictícia de recrutamento de desenvolvedores, utilizando HTML e CSS.

## 🎯 Objetivo

Aplicar os conhecimentos adquiridos no curso de HTML e CSS da Alura, criando uma página responsiva e visualmente atraente, com foco em boas práticas de desenvolvimento front-end.

## 🧪 Tecnologias Utilizadas

- HTML5
- CSS3

## 📁 Estrutura do Projeto

- `index.html`: Estrutura principal da página.
- `style.css`: Arquivo de estilos.
- `imgs/`: Pasta com as imagens utilizadas no projeto.

## 📚 O que foi aprendido

- Estruturação semântica com HTML.
- Estilização de componentes com CSS.
- Criação de layouts responsivos utilizando media queries.
- Boas práticas no desenvolvimento de páginas web.
